Описание программы
Book Tracker - это GUI-приложение для управления списком прочитанных книг. Приложение позволяет добавлять книги, фильтровать их по жанру и количеству страниц, сохранять и загружать данные в формате JSON, а также удалять отдельные записи или весь список целиком.

Возможности:
✅ Добавление книг с указанием названия, автора, жанра и количества страниц
✅ Фильтрация книг по жанру (Роман, Фантастика, Детектив, и др.)
✅ Фильтрация по количеству страниц (>200, >300, >500)
✅ Сохранение данных в JSON-файл
✅ Загрузка данных из JSON-файла
✅ Удаление выбранной книги
✅ Очистка всего списка книг
✅ Валидация вводимых данных (проверка пустых полей, числовое значение страниц)
Инструкция по запуску
Требования:
Python 3.6 или выше
Установленная библиотека Tkinter (обычно идёт в комплекте с Python)
Запуск:
Откройте терминал или командную строку
Перейдите в директорию проекта:
cd bookTracker
Запустите приложение:
python main.py
Примеры использования
Добавление книги:
Заполните все поля формы:
Название книги: "Мастер и Маргарита"
Автор: "Михаил Булгаков"
Жанр: Выберите из списка (например, "Роман")
Количество страниц: 480
Нажмите кнопку "Добавить книгу"
Книга появится в списке внизу окна
Фильтрация:
По жанру: Выберите жанр из выпадающего списка в разделе "Фильтры" и нажмите "Применить фильтр"
По страницам: Выберите критерий (">200", ">300", ">500") и нажмите "Применить фильтр"
Можно комбинировать оба фильтра одновременно
Для сброса нажмите "Сбросить"
Сохранение и загрузка:
Нажмите "Сохранить в JSON" - данные сохранятся в файл books_data.json
Нажмите "Загрузить из JSON" - данные загрузятся из файла books_data.json
Удаление:
Выберите книгу в списке и нажмите "Удалить выбранную"
Для удаления всех книг нажмите "Очистить всё" и подтвердите действие

Валидация данных
Приложение проверяет:

Все поля должны быть заполнены
Количество страниц должно быть положительным числом
При удалении необходимо выбрать книгу из списка
При очистке всего списка требуется подтверждение
Структура проекта
bookTracker/

├── README.md         # Документация проекта
